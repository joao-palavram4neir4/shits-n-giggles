README
                      
O  jogo foi feito utilizando o RPG MAKER
O RPG Maker é uma ferramenta de desenvolvimento de jogos focada na criação de RPGs 2D de forma acessível e intuitiva. Ele permite que pessoas criem jogos sem precisar construir um motor do zero, utilizando um sistema visual de mapas, eventos e comandos pré-configurados.

Com ele, é possível desenvolver praticamente todos os elementos de um jogo, como exploração de cenários, diálogos entre personagens, sistema de missões, cutscenes e interações. Isso torna o programa bastante popular tanto para iniciantes quanto para desenvolvedores independentes que desejam focar mais na parte criativa e narrativa do projeto :3

Sobre este jogo

Este projeto foi desenvolvido utilizando o RPG Maker, com foco em criar uma experiência de aventura e exploração baseada no tema Agrinho. O jogo foi pensado para apresentar uma narrativa simples, mas envolvente, onde o jogador avança por meio de interações com personagens, exploração de mapas e progressão através de eventos.

O objetivo do projeto é proporcionar uma experiência leve e acessível, destacando a construção de ambientes e a forma como a história é apresentada ao jogador. Cada área foi criada para contribuir com a imersão e reforçar a proposta do jogo, incentivando o progresso de maneira natural ao longo da jornada :3

## ⚠️ Aviso

Este jogo não possui fins lucrativos e foi desenvolvido apenas com finalidade educacional e de aprendizado.  

Algumas músicas utilizadas no projeto não são de autoria própria e pertencem aos seus respectivos criadores, sendo usadas apenas para compor a atmosfera do jogo sem qualquer intenção comercial.
